Python Scripting¶

I assume you have a working knowledge of Python. websharpstudios implements two Python modules – one great huge one called websharpstudios which provides access to as much of websharpstudios’s functionality as I’ve had time to write, and one tiny one called psMat which provides quick access to some useful transformations expressed as PostScript matrices.

In python terms websharpstudios embeds python. It is possible to build websharpstudios so that it is also a python extension.

Command line convenience¶

For convenience, Python commands given as a -c argument on the command line have the following code prepended:

from sys import argv; from websharpstudios import *

Hence, the trivial script to convert a font can be written:

websharpstudios -c 'open(argv[1]).generate(argv[2])'

Trivial example¶

import websharpstudios                                 #Load the module
amb=websharpstudios.open("Ambrosia.sfd")               #Open a font
amb.selection.select(("ranges",None),"A","Z")    #select A-Z
amb.copy()                                       #Copy those glyphs into the clipboard

n=websharpstudios.font()                               #Create a new font
n.selection.select(("ranges",None),"A","Z")      #select A-Z of it
n.paste()                                        #paste the glyphs above in
print n["A"].foreground                          #test to see that something
                                                 #  actually got pasted
n.fontname="NewFont"                             #Give the new font a name
n.save("NewFont.sfd")                            #and save it.

websharpstudios as a python extension¶

In addition to embedding Python, websharpstudios typically installs a Python module accessible to the system’s python executable, which can be accessed using:

>>> import websharpstudios
