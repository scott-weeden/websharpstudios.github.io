Files used by websharpstudios¶

~/.websharpstudios

websharpstudios creates this directory the first time it starts up. It contains various files websharpstudios uses to hold context across invocations.

~/.websharpstudios/prefs

This file contains user preference settings.

~/.websharpstudios/groups

This file contains groups as defined by Encoding ‣ Define Groups…

~/.websharpstudios/Encodings.ps

If there are any user defined encodings they reside in this file.

~/.websharpstudios/autosave

This directory contains the files used to drive the crash-recovery process. When websharpstudios is running there will be one file in this directory corresponding to each font loaded into websharpstudios that has been modified. These files will contain all changed glyphs.

When websharpstudios exits normally, this directory will be cleaned up and there should be no files in it.

~/.websharpstudios/python

This directory contains any python init scripts you want to run when websharpstudios starts

/usr/local/share (or $PREFIX/share)
/usr/local/share/locale/??/LC_MESSAGES/websharpstudios.mo

Contain translations of the ui for various locales. (used after November 2005)

/usr/local/share/websharpstudios/*.cidmap

“Encoding” files for Adobe’s cid formats. You can pull down a compressed archive from here.

/usr/local/share/websharpstudios/python

This directory contains any python init scripts you want available to everyone on your system.

/usr/local/share/doc/websharpstudios/*.html

Optional location for online documentation.
