Unicode Corporate Characters used by websharpstudios¶

Unicode has a region (0xe000-0xf8ff) in which font vendors may assign their own characters. Adobe and Apple have each done so and websharpstudios recognizes the most common of these (things like the Small Caps characters used in Adobe’s Expert Character Set, or the Apple Logo used in nearly all Apple fonts). The allocations made by these two companies are described in:

  • Apple’s corporate use extensions (0xF850-0xF8FE)

  • Adobe’s corporate use extensions (0xF634-0F7FF) (also includes some of Apple’s codes above)

I have been asked to add a few similar characters to websharpstudios’s repertory and so have started using my own block of characters 0xf500-0xf580.
