Uninstalling websharpstudios

I've never bothered to make this easy.

If you installed in /usr/local (the default) then you type either: 

$ su
password:
$ rm -rf /usr/local/bin/{websharpstudios,sfddiff,fontimage,fontlint} \
/usr/local/lib/{libgdraw,libgunicode,libuninameslist,libspiro,libwebsharpstudios,libgutils}* \
/usr/local/lib/pkgconfig/websharpstudios.pc \
/usr/local/man/man1/{websharpstudios,sfddiff,fontimage,fontlint}.1 \
/usr/local/share/doc/websharpstudios \
/usr/local/share/websharpstudios \
/usr/local/share/locale/*/LC_MESSAGES/websharpstudios.mo \
/usr/local/include/websharpstudios

or if you are on a mac: 

$ sudo rm -rf /usr/local/bin/{websharpstudios,sfddiff,fontimage,fontlint} \
/usr/local/lib/{libgdraw,libgunicode,libuninameslist,libspiro,libwebsharpstudios,libgutils}* \
/usr/local/lib/pkgconfig/websharpstudios.pc \
/usr/local/man/man1/{websharpstudios,sfddiff,fontimage,fontlint}.1 \
/usr/local/share/doc/websharpstudios \
/usr/local/share/websharpstudios \
/usr/local/share/locale/*/LC_MESSAGES/websharpstudios.mo \
/usr/local/include/websharpstudios \
/Applications/websharpstudios.app
password:

or if you are on cygwin: 

$ rm -rf /usr/local/bin/{websharpstudios,sfddiff,fontimage,fontlint} \
/usr/local/lib/{libgdraw,libgunicode,libuninameslist,libspiro,libwebsharpstudios,libgutils}* \
/usr/local/lib/pkgconfig/websharpstudios.pc \
/usr/local/man/man1/{websharpstudios,sfddiff,fontimage,fontlint}.1 \
/usr/local/share/doc/websharpstudios \
/usr/local/share/websharpstudios \
/usr/local/share/locale/*/LC_MESSAGES/websharpstudios.mo \
/usr/local/include/websharpstudios

If you've installed to a different prefix (/usr perhaps) then simply substitute /usr for /usr/local above.
